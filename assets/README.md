# Assets Folder

This folder contains images and other media files for your portfolio website.

## Required Images

### Profile Image

- **File name:** `profile.jpg` (or `profile.png`)
- **Size:** 400x400px minimum (square)
- **Format:** JPG or PNG
- **Purpose:** Main profile photo displayed in the hero section

**Tips for best results:**

- Use a professional headshot with good lighting
- Ensure your face is centered
- Use a clean background (solid color or blurred)
- Image should be at least 400x400px for sharp display on all devices

### Optional Images

You can add additional images for:

- Case study screenshots
- Project thumbnails
- Company logos
- Certificates or awards

## How to Add Your Profile Photo

1. **Prepare your image:**

   - Resize to at least 400x400px (square format works best)
   - Optimize for web (compress to keep file size under 500KB)

2. **Add to this folder:**

   - Save as `profile.jpg` or `profile.png`
   - Place it in this `assets/` folder

3. **If using different name:**
   - Update line in `index.html`:
   ```html
   <img src="assets/profile.jpg" alt="Asim Riaz" class="profile-image" />
   ```

## Recommended Tools

**Free online tools for image editing:**

- **Remove background:** https://www.remove.bg/
